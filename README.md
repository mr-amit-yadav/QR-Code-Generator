# Professional QR Code Generator

## 📌 Overview

AI QR Code Generator Pro is an interactive web application built using **Streamlit** that allows users to generate customized QR codes instantly. The app supports different content types and provides styling and error correction options for better flexibility and usability.

---

## 🎯 Features

* 🔗 Generate QR codes for URLs and other content types
* 🎨 Custom styling (colors, shapes, and design)
* ⚙️ Adjustable error correction levels
* 📥 Download generated QR codes
* ⚡ Fast and user-friendly interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/QR-Generator.git
cd QR-Generator
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```
